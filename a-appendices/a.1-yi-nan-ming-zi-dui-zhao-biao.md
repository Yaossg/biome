# A.1 疑难名字对照表

## 噪声

相比而言，我更赞同 yarn 的名字，因为被 yarn 冠以“八度”之名的噪声生成器，内部（underlying）都是对应的普通噪声生成器的数组，对这些噪声分形取样得到新的噪声。

|  | mcp | yarn |  |
| :--- | :---: | :---: | :--- |
| 改进的 | Improved | Perlin | 柏林 |
| 简单的 | Simplex | Simplex | 简单 |
| 八度的 | Octaves | OctavePerlin | 八度柏林 |
| 柏林的 | Perlin | OctaveSimplex | 八度简单 |
